# Symbols

- \(\theta\): frozen base model parameters
- \(\phi\): repair operator parameters
- \(l\): layer index
- \(h\): head index
- \(H\): number of heads
- \(T\): cache length (past tokens)
- \(d\): head dimension
- \(S\): KV cache state over protected layers
- \(\mathcal{C}\): corruption family
- \(\Pi\): corruption mixture distribution
- \(R_\phi\): repair/stability operator
- \(z(\cdot)\): next-token logits
- \(\epsilon\): corruption severity
- \(\delta\): additive state perturbation used for stability evaluation
